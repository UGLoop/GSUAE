## ğŸ“¦ Deployment

### Deploy to Vercel

1. Fork or clone this repository
2. Connect your GitHub account to Vercel
3. Import this repository in Vercel
4. Deploy with default settings

### Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/golo-consultancy-website.git

# Navigate to project directory
cd golo-consultancy-website

# Install dependencies (optional, for local server)
npm install

# Run local development server
npm run dev

# Open browser to http://localhost:3000
```

## ğŸ”§ Configuration

### Custom Domain Setup (Vercel)

1. Go to your Vercel project settings
2. Navigate to "Domains"
3. Add your custom domain: goloconsultancy.com
4. Update your domain's DNS settings:
   - Add A record pointing to Vercel's IP
   - Add CNAME record for www subdomain
